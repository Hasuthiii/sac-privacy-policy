Política de Privacidad

Última actualización: Febrero 2025

1. Información que recopilamos

Sistemas de gestión SAC recopila la siguiente información:

  • Datos de cuenta: nombre de usuario y contraseña para autenticación
  • Imágenes: fotografías tomadas con la cámara del dispositivo para documentación de coladas y recolecciones
  • Datos operativos: información relacionada con coladas TRF DROSS y recolecciones BUPA

2. Uso de permisos

Cámara: Se utiliza únicamente para capturar fotografías como evidencia de coladas y recolecciones. Las imágenes se almacenan de forma segura en nuestros servidores.

3. Cómo usamos la información

La información recopilada se utiliza exclusivamente para:

  • Gestión de operaciones industriales
  • Seguimiento de coladas y recolecciones
  • Generación de reportes internos

4. Seguridad de datos

Todos los datos se transmiten de forma cifrada mediante HTTPS. Implementamos medidas de seguridad para proteger la información contra acceso no autorizado.

5. Compartir información

No compartimos información personal con terceros. Los datos son de uso interno exclusivo de SAC Company.

6. Uso empresarial

Esta aplicación es de uso exclusivo para empleados y personal autorizado de SAC Company.
